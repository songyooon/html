미디어 관련 요소

이미지 5개 이상 - jpg(jfif), png, gif, webp, svg

동영상 3개 이상 - mp4, webm, ogv

  • video 요소의 태그 속성

    src : 해당 비디오 파일의 경로(위치와 파일명)

    poster : 대기화면(썸네일) 이미지의 파일 경로

    controls : 비디오 재생 제어 패널 표시

    autoplay : 자동재생

    muted : 무음처리(자동 재생시 반드시 필요함)

    loop : 반복 재생

    width : 폭, height : 높이

    아래의 것을 크로스브라우징이라고 부름

  • 브라우저마다 재생가능한 파일

    mp4, webm, ogv 파일이 브라우저마다 각기 다르게 재생 기능 제공됨

    브라우저MP4WebMOgv
    Internet Explorer YES No No
    Chrome YES Yes Yes
    Firefox YES Yes Yes
    Safari YES No No
    Opera YES Yes Yes

사운드 3개 이상 - mp3, ogg, wav

  • audio 요소의 태그 속성

    src : 해당 오디오 파일의 경로(위치와 파일명)

    controls : 오디오 재생 제어 패널 표시

    autoplay : 자동재생

    muted : 무음처리(자동 재생시 반드시 필요함)

    loop : 반복 재생

  • 브라우저 마다 재생가능한 파일

    mp3, wav, ogg 파일이 브라우저 마다 각기 다르게 재생 기능 제공됨

    브라우저MP3WAVOGG
    Internet ExplorerYESNoNo
    ChromeYESYesYes
    FirefoxYESYesYes
    SafariYESYesNo
    OperaYESYesYes

  • ※ audio/video의 내장 관련 태그로 embed나 object를 활용했었으나 html5 웹 표준에서는 플래시와 같은 플러그인의 사용 금지로 잘 사용하지 않음

    embed/object 의 태그 속성

    src 또는 data : 해당 동영상, 오디오, 플래시, pdf 파일의 경로

    width : 폭, height : 높이

    loop : 반복 재생

    autostart : 자동재생

    mute : 무음처리(자동 재생시 반드시 필요함)

    hidden : 오디오 파일과 같이 화면에 표시하지 않는 경우 숨김

    volume : 볼륨(소리)의 크기 지정

    showcontrols : 미디어 재생 제어 패널 표시

    type : 해당 video나 audio와 같은 미디어의 타입을 선언
